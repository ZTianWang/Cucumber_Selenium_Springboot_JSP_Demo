**Technologies**

- Junit

- SpringBoot

to run the web application.  
`./gradlew bootRun`.   
then go to http://localhost:8080/login         
username: user1 password:password

to run the cucumber tests.   
` ./gradlew cucumber`

to run backend tests.  
` ./gradlew test`

to run both.   
` ./gradlew test cucumber`
